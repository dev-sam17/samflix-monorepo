# @samflix/tsconfig

Shared TypeScript configurations for Samflix monorepo.

## Usage

### For Node.js/Backend apps:

```json
{
  "extends": "@samflix/tsconfig/node.json"
}
```

### For Next.js apps:

```json
{
  "extends": "@samflix/tsconfig/nextjs.json"
}
```

### For React apps:

```json
{
  "extends": "@samflix/tsconfig/react.json"
}
```
